Execute Jobs of a JobCollection

Usage

doJobCollection(jc, output = NULL)

Arguments

jc
[JobCollection] Either an object of class “JobCollection” as returned by makeJobCollection or a string with the path to file containing a “JobCollection” as RDS file (as stored by submitJobs).
output
[character(1)] Path to a file to write the output to. Defaults to NULL which means that output is written to the active sink.

Value

[character(1)]: Hash of the JobCollection executed.

Description

Executes every job in a JobCollection. This function is intended to be called on the slave.

Examples

tmp = makeRegistry(file.dir = NA, make.default = FALSE)

Sourcing configuration file '~/.batchtools.conf.R' ...

batchMap(identity, 1:2, reg = tmp)

Adding 2 jobs ...

jc = makeJobCollection(1:2, reg = tmp)
doJobCollection(jc)
