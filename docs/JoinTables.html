Inner, Left, Right, Outer, Semi and Anti Join for Job Tables

Usage

ijoin(x, y)

ljoin(x, y)

rjoin(x, y)

ojoin(x, y)

sjoin(x, y)

ajoin(x, y)

Arguments

x
[data.frame | integer] Either a data.table/data.frame with integer column “job.id” or an integer vector of job ids.
y
[data.frame | integer] Either a data.table/data.frame with integer column “job.id” or an integer vector of job ids.

Value

[data.table] with key “job.id”.

Description

These helper functions perform join operations on job tables. They are basically one-liners with additional argument checks for sanity. See http://rpubs.com/ronasta/join_data_tables for a overview of join operations in data table or alternatively dplyr's vignette on two table verbs.

Examples

# Create two tables for demonstration
tmp = makeRegistry(file.dir = NA, make.default = FALSE)

Sourcing configuration file '~/.batchtools.conf.R' ...

batchMap(identity, x = 1:6, reg = tmp)

Adding 6 jobs ...

x = getJobPars(reg = tmp)
y = findJobs(x >= 2 & x <= 5, reg = tmp)
y$extra.col = head(letters, nrow(y))

# Inner join: similar to intersect() on ids, keep all columns of x and y
ijoin(x, y)


   job.id x extra.col
1:      2 2         a
2:      3 3         b
3:      4 4         c
4:      5 5         d




# Left join: use all ids from x, keep all columns of x and y
ljoin(x, y)


   job.id extra.col x
1:      1        NA 1
2:      2         a 2
3:      3         b 3
4:      4         c 4
5:      5         d 5
6:      6        NA 6




# Right join: use all ids from y, keep all columns of x and y
rjoin(x, y)


   job.id x extra.col
1:      2 2         a
2:      3 3         b
3:      4 4         c
4:      5 5         d




# Outer join: similar to union() on ids, keep all columns of x and y
ojoin(x, y)


   job.id x extra.col
1:      1 1        NA
2:      2 2         a
3:      3 3         b
4:      4 4         c
5:      5 5         d
6:      6 6        NA




# Semi join: similar to intersect() on ids, keep all columns of x
sjoin(x, y)


   job.id x
1:      2 2
2:      3 3
3:      4 4
4:      5 5




# Anti join: similar to setdiff() on ids, keep all columns of x
ajoin(x, y)


   job.id x
1:      1 1
2:      6 6
