Run OS Commands on Local or Remote Machines

Usage

runOSCommand(sys.cmd, sys.args = character(0L), nodename = "localhost",
  debug = getOption("batchtools.debug", FALSE))

Arguments

sys.cmd
[character(1)] Command to run.
sys.args
[character] Arguments for sys.cmd.
nodename
[character(1)] Name of the SSH node to run the command on. If set to “localhost” (default), the command is not piped through SSH.
debug
[logical(1)] If TRUE, the commands send to the OS are printed.

Value

[named list] with “exit.code” (integer) and “output” (character).

Description

This is a helper function to run arbitrary OS commands on local or remote machines. The interface is similar to system2, but it always returns the exit status and the output.

Examples

# runOSCommand("ls")
# runOSCommand("ls", "-al")
# runOSCommand("notfound")
## ---------------------------------------------
